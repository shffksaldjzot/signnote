1. 협력업체 행사목록(홈)에서 행사카드 옆에 3점 아이콘 누르면 행사참가취소가 가능해야돼 그때 참가 취소하시겠습니까? 팝업 한번 뜨고 본인 비밀번호 입력해야 참가 취소 되는방식으로 만들자.
2. 주관사 행사목록(홈)에서 3점 버튼 눌러서 행사 편집, 행사 삭제 기능 만들어야돼고 행사 편집 누르면 행사등록이랑 똑같은 페이지로 넘어가서 편집할수 있게 만들면돼. 삭제할때는 주관사(본인)계정 비밀번호 한번더 입력하도록 해줘.
3. 주관하 행사 등록/편집 페이지에서 행사 기간을 취소가능기간(선택) 위로 옮겨줘.
4. 어드민 파트너관리 탭을 사용자 관리로 이름 바꾸줘.
5. 관리자-행사관리-주관사로 정렬할수 있는 기능 만들어줘.
6. 고객도 로그인 할때마다 행사코드 입력하는 화면이 나오는데 이것도 이미 참여한 행사가 있는 고객은 홈화면으로 바로 이동.
7. 어드민이 회원 탈퇴 시키는 기능도 만들어줘.
8. 로그인 페이지에서 비밀번호를 잊으셨나요? 버튼 만들어서 기능 붙일 준비해줘 이거는 나중에 이메일로 인증해서 비밀번호 찾을건데 지금 이메일시스템붙이지는 않을거야.
