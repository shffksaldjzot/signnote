✅2.협력업체가 계약서다운로드하려고하면 다룬로드 기능은 준비중이라고 하는데 바로 다운로드 할수 있게 해줘. 아까 말한 내용들 전부 넣어서. 협력업체뿐만 아니라 관리자 주관사 고객 모두 말이야. 계약서에 작성될 내용은 행사이름 단지명 고객정보 전부 협력업체정보 전부 주관사이름 결제내용전부야. → 처리됨 (업체 계약 상세 화면에 RepaintBoundary 캡처 다운로드 구현, "준비중" 제거)
✅3.모든 알림에 전부 읽음으로 표시 기능 만들어줘. → 처리됨 (주관사 모바일/웹 알림 탭에 "전부 읽음으로 표시" 버튼 추가, 기존 고객/업체 알림은 이미 있음)
✅1. 어드민페이지에서 업체관리에서 관리자는 삭제 불가능하게 만들어줘 아니면 리스트에서 빼던지. → 처리됨 (업체관리 리스트에서 ADMIN 역할 제외)
✅2. 관리자가 행사관리에서 계약현황탭에서 계약건 클릭하면 계약상세페이지가 나오고 다운로드도 가능해야돼 계약서 다운로드 말이야. → 처리됨 (계약 행 클릭 → 상세 다이얼로그 + RepaintBoundary 캡처 다운로드)
✅3. 관리자 업체 관리에서 신규 업체가 가입신청을 하면 좌측 메뉴바에 업체관리 옆에 빨강색 알림에 숫자 써줘. → 처리됨 (사이드바 미승인 업체 수 빨간 배지 표시)
✅4. 주관사가 품목관리에서 품목을 추가할수는 있는데 뺄수는 없네 품목 삭제할수 있는 기능도 넣어줘. 그리고 품목 이름 변경도 할수 있게 해줘. → 처리됨 (백엔드 DELETE API + 프론트 이름변경/삭제 버튼 추가)
✅5. 주관사가 품목관리에서 업체 배정하면 배정된 아코디언 순으로 정렬되는데 이거 아코디언위치는 고정해줘 내가 위치 이동 기능으로 정리할거야 그리고 화살표로 위치 이동하지 말고 스마트폰 방식으로 꾹 누르고 있으면 짚을수 있고 내가 원하는데에 떨어뜨리면 거기로 옮겨지는 방식으로 위치이동하고 싶어 우리 스마트촌 바탕하면 아이콘 옮기는 방식으로 말이야 가능할지 확인해보자. → 처리됨 (ReorderableListView 드래그앤드롭 방식으로 변경, 화살표 제거)
✅6. 이미 한번 품목에 협력업체를 지정하면 맨위에 있는 미배정이 다시 선택이 안돼 배정된 업체가 없도록 공백으로 비우고 싶을수도 있으니 미배정도 선택되도록 해줘. → 처리됨 (미배정 선택 시 unclaimProduct 호출)
✅7. 어드민 고객 관리에서 행사 선택안하면 전체 고객이 다 보여야돼 맨위에 "행사를 선택해주세요"를 "전체"로 바꿔줘 그럼 전체로 되어있을땐 모든 고객 다 보이게. → 처리됨 (기본값 "전체" + 전체 고객 API 조회)
✅8. 고객 가입후에 첫 로그인하자마가 행사코드 입력하고 동호수, 타입 선택하는 칸에서 동호수 필드가 뭔가 눈에 잘안띄네 필수 입력해야돼는데 이거 동호수 입력 눈에 잘띄도록해줘. → 처리됨 (파란 테두리+배경 + 라벨 + 힌트텍스트 + 필수표시)
✅9. 고객 첫 랜딩페이지에서 구매품목리스트가 쭉나오는데 그때 아코디언이 접혀 있는게 기본이야. 그리고 아코디언 우측 상당에 펼치는 화살표도 있는게 맞긴한데. "외 2개 더보기" 이부분 클릭하면 열려야지 아코디언이. 아니면 우측상단 화살표가 아래쪽으로 오던지. → 처리됨 (기본 접힘 + "외 N개 더보기" 클릭 시 펼침)
✅10. 고객이 계약이후에 계약서 다운받았을때 어떤식으로 입금했는지 카드일땐 카드결제정보 전부, 계좌이체일땐 계좌이체 내역이 나와야지. 아직 결제를 구현하지 않았어도 여기에 결제정도가 들어간다라는 입력칸은 만들어줘. 그리고 영수증 글씨중에 회색톤은 너무 흐린것같아 좀더 진하게 해줘. → 처리됨 (결제정보 영역 추가 + 회색 글씨 진하게)
✅11. 고객이 결제하는 방식에서 간편결제는 옆에(준비중)이라고 써줘. → 처리됨 (간편결제 비활성화 + "(준비중)" 텍스트)
