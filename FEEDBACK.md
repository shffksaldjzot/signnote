7. 협력업체가 품목추가하기에서 타입선택할때 앞에 v자 생기면서 배지가 커지는데 그렇게 하지말고 그냥 색깔만 바뀌게 해줘.
8. 협력업체가 품목에 배정되어있지 않은 협력업체는 품목 추가하기 버튼을 눌렀을때 일단 품목 추가하기 페이지로 넘어가지는게 아니고 배정된 품목이 없습니다.라고 알림이 뜨게 해줘.
